Este sitio web lo construi con Next.js y voy a hablar un poco sobre como configure MDX para crear el blog. Para lograr esto, me fue de gran ayuda este [articulo](https://sergiodxa.com/articles/using-mdx-with-remote-content) publicado por [Sergio Xalambri](https://twitter.com/sergiodxa)

Entonces, primero hay que instalar **MDX** en nuestro proyecto de **Next.js**. Aqui estan las [instrucciones](https://mdxjs.com/getting-started/next), es realmente facil

Despues, instalar el paquete [next-mdx-remote (Github)](https://github.com/hashicorp/next-mdx-remote). Este paquete permite renderizar archivos MDX de manera dinamica. Por los ejemplos que he encontrado en Github, entiendo que usualmente hay que crear una pagina en Next.js para cada archivo MDX que se quiera mostrar. Ademas, los archivos MDX se guardan en el mismo repositorio del proyecto. next-mdx-remote permite renderizar archivos MDX de manera dinamica, que se pueden encontrar en un lugar diferente al repositorio del proyecto, por ejemplo, un CMS, un CDN de archivos estaticos, o cualquier otro lugar. Asi que por performance y otras razones mas [(aqui las pueden leer)](https://github.com/hashicorp/next-mdx-remote#background--theory), este paquete me fue muy util para lograr lo que queria

Ahora, un ejemplo de como uso **next-mdx-remote** en una pagina de Next.js que carga un blog post de acuerdo a un parametro de URL. `/blog/[slug]`

Hay que importar algunas funciones de **next-mdx-remote**

```
// Importar las dependencias
import hydrate from "next-mdx-remote/hydrate";
import renderToString from "next-mdx-remote/render-to-string";
```

Importar los componentes que queremos usar dentros nuestros archivos MDX

```
import * as BlogPostsComponents from "~/components/pages/blog-post/BlogPostsComponents";
```

Este es el componente de nuestra pagina de Next.js, el cual va a renderizar el archivo MDX

```
function BlogPostPage({ post, content }: Record<string, any>): any {
  const mdxContent = hydrate(content, { components: BlogPostsComponents });

  return (
    <Page>
      <MainLayout
        breadcumb={[
          { text: "Inicio", url: Routes.HOME },
          { text: "Blog", url: Routes.BLOG() },
          { text: post.title, url: Routes.BLOG(post.slug) },
        ]}
        title={post.title}
      >
        <BlogPostContent content={mdxContent} />
      </MainLayout>
    </Page>
  );
}

export default BlogPostPage;
```
