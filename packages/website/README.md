# website

My personal website

## Tech stack

- TypeScript
- React
- Next.js
- Tailwind CSS
- MDX

## Setup

1. `git clone https://github.com/diegofrayo/website.git`
1. `cd website`
1. `npm i`
1. `npm run dev`
1. Create a `.env` file
  ```
  NEXT_PUBLIC_WEBSITE_URL=http://localhost:3000
  ```
---

Hosted on [Vercel](https://vercel.com)
